# EduVerse Backend

Backend API for EduVerse.

## Tech Stack

- NestJS
- TypeScript
- MySQL (TypeORM)
- Docker

## Local Development

1. Install dependencies:

```bash
npm install
```

2. Configure environment variables in `.env`.

3. Start development server:

```bash
npm run start:dev
```

Default local API URL: `http://localhost:8081`
